drag-and-drop 拖放

  • pDraggable : 設定可拖曳的行為，拖拉的來源資料。
  • Draggable attributes :dragEffect(定義滑鼠的樣式)、dragHandle、pDraggableDisabled
  • Draggable event :onDragStart、onDrag、onDragEnd。
  • pDroppable : 設定可放置的行為，放置的目的地資料，使用陣列做儲存。
  • pDroppable attributes : dropEffect、pDroppableDisabled。
  • pDroppable event : onDragEnter、onDrop、onDragLeave。
