MenuModel

基本上 menu 都使用 MenuItem 的資料模型，除了 MegaMenu 是使用 MegaMenuItem。

MenuItem 常用的欄位，更多欄位請看官網文檔
  • label : 顯示文字
  • icon : 要顯示的 icon ('pi pi-plus')
  • routerLink : Angular 的 routing
  • queryParams : queryParams
  • queryParams : fragment
  • url : 另開外部連結
  • visible : 是否能看見
  • disabled : 是否禁用
  • command : 點擊後執行事件
  • escape : true 的時候，label 可以撰寫 html 相關 tag
  • items : 子選項，MenuItem 陣列
